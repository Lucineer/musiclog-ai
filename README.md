# musiclog-ai

![Cocapn Vessel](https://img.shields.io/badge/cocapn-vessel-purple) ![License](https://img.shields.io/badge/license-MIT-blue)

**Your AI music companion — playlists, discovery, and sonic exploration.**

Part of the [Cocapn fleet](https://github.com/Lucineer).

## Features
- AI-powered playlist generation
- Music discovery across genres
- Listening history and preferences
- BYOK (Bring Your Own Keys)

## Deploy

Fork, add your API key as a CF secret, deploy with wrangler.
